How Message Archive Utility works

Create an unencrypted local iPhone backup, import messages from that backup, then search and export your archive.

Before you start

What you need

  • You need a Mac.
  • You need a local iPhone backup.
  • This beta requires the backup to be unencrypted.
  • The app does not pull messages from iCloud.
  1. 1

    Back up your iPhone on your Mac

    Create a local iPhone backup in Finder and leave local backup encryption off for this beta.

  2. 2

    Import messages from the backup

    Choose the backup on this Mac. The app imports the messages it can read and builds a local archive.

  3. 3

    Browse and search conversations

    Open conversation threads, search for names or phrases, and review matching messages.

  4. 4

    Export PDF, Excel, or CSV files

    Create readable files from the archive views the app supports.

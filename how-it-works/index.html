How Message Archive Utility works

Create an unencrypted local iPhone backup, import messages from that backup, then search and export your archive.

Before you start

What you need

  • You need a Mac.
  • You need a local iPhone backup.
  • This beta requires the backup to be unencrypted.
  • The app does not pull messages from iCloud.
  1. 1

    Connect your iPhone to your Mac

    Use a cable, unlock your iPhone if prompted, and choose to trust this Mac.

  2. 2

    Create the backup in Finder

    Select your iPhone in the Finder sidebar, choose a local backup, and leave "Encrypt local backup" unchecked for this beta/import.

  3. 3

    Run the backup, then import

    After Finder finishes the backup, return to Message Archive Utility and import messages from that backup.

  4. 4

    Search and export

    Browse conversations, search words or names, and export PDF, Excel, or CSV files from supported archive views.

Backup note

Keep the unencrypted backup limited to the import step.

If you normally use encrypted iPhone backups, create a temporary unencrypted backup for this import, then delete it or return to your normal backup settings afterward.

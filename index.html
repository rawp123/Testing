What it does

A focused Mac utility for old messages.

Import from a local backup

Use an unencrypted iPhone backup stored on this Mac as the message source.

Search conversations

Browse threads and search old messages by names, words, and phrases.

Export useful copies

Create PDF, Excel, or CSV files from the archive views the app supports.

First import

Start with a local iPhone backup.

The backup step happens in Finder. After that, Message Archive Utility imports the messages it can read and keeps the archive on this Mac.

  1. 1

    Back up your iPhone in Finder

    Connect your iPhone, open Finder, and create a local backup on this Mac.

  2. 2

    Leave encryption off for this import

    For the current beta, make sure "Encrypt local backup" is not selected.

  3. 3

    Import, search, and export

    Choose the backup in the app, then search conversations and export PDF, Excel, or CSV files.

What you can export.

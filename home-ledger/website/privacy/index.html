Privacy

Your records stay local.

Home Basis Tracker is designed as a local record binder. It does not create an account, upload records, sync to a cloud service, add telemetry, or call third-party APIs for your records.

Mac desktop app

In the Mac app, structured records are stored in app-managed local storage. Attached files are copied into an app-managed documents folder so the app can keep them connected to your property, project, and expense records.

Browser version

In the browser version, structured records use the current browser profile with localStorage. Attached document files are stored in that browser profile with IndexedDB.

Backups are your responsibility

Local storage is private and convenient, but it is not a backup by itself. Deleting app data, clearing browser data, changing browser profiles, or using private browsing can remove records and attachments.

Full backup files can contain sensitive home, vendor, amount, note, receipt, invoice, photo, and document contents. Treat them like private records and store them somewhere you trust.

What the app does not add

  • No account requirement.
  • No cloud sync workflow.
  • No record upload workflow.
  • No telemetry or analytics code.
  • No third-party APIs for your records.

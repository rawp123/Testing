How it works

Build the record as the work happens.

Home Ledger keeps the basic steps clear: add a home, group work into projects, add costs, attach supporting files, and export summaries when you need them.

1

Add a property

Start with home details such as address, purchase date, purchase price, and notes.

2

Create projects

Group improvements such as kitchen remodels, roof work, HVAC, landscaping, windows, permits, or repairs.

3

Add expenses

Enter the date, vendor, description, amount, category, cost type, receipt/file status, and notes.

4

Attach documents

Keep receipts, invoices, permits, photos, and related documents connected to the cost or project they support.

5

Export or back up

Create a CSV, print a packet, or save a full backup for long-term safekeeping.

Organized around real home projects

Every expense belongs to a property and can also belong to a project. That makes it easier to review one renovation, compare work across years, or pull a complete property summary later.

  • Home details for one or multiple properties.
  • Project history with contractor, category, dates, and notes.
  • Expense details with cost type and receipt/file status.
  • File details and optional attachments.

Built for records, not advice

Home Ledger organizes records. It does not give tax, legal, or accounting advice.

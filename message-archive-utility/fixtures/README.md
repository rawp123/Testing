# Fixtures

Use this folder only for safe Message Archive Utility QA fixtures.

Private archives belong in:

```text
fixtures/private-archives/
```

That folder is ignored by Git. Do not commit real phone backups, message databases, attachments, exports, contacts, phone numbers, message text, screenshots, or troubleshooting logs that contain private message data.

The backend test fixtures under `backend/tests/fixtures/` are synthetic and safe to commit.

Availability

Private beta access is limited.

Message Archive Utility is being prepared for a small private beta before any public download is offered. The app is a local-first Mac utility, and messages stay on the user's Mac.

Current status

The app is not available for public download yet. A signed Mac build is being reviewed through limited testing before broader release details are posted.

Private beta

Private beta access may be shared with named testers who understand that the app is still being checked carefully. There is no open download link on this website.

How access works

Beta delivery is handled privately outside the public website. The app file is not stored in this site repository and is not exposed through a public download path.

What testers should expect

Testers should use only messages they are comfortable testing with, keep their own exports in a safe place, and report any confusing steps or broken behavior.

Before using the app

Read the walkthrough, the privacy notes, and the FAQ. They explain the local backup workflow, export formats, and current limits.

Public release

Public download and pricing details will be added only after the release flow is ready. Until then, this page is informational and does not provide a download.

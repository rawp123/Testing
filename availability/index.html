Availability

Private beta access

Message Archive Utility is available only to named private-beta testers right now. The current build is Mac only, and this site does not offer a public download or payment link.

Current status

Message Archive Utility is in private beta. Test builds are shared directly with named testers.

What testers receive

  • A Mac app build for private testing.
  • Basic install and import instructions.
  • PDF, Excel, and CSV export options.

What testers need

  • A Mac.
  • An iPhone backup stored locally on that Mac.
  • An unencrypted backup if importing messages in this beta.

What to report

  • Install or first-open issues.
  • Confusing backup steps.
  • Import, search, or export problems.
  • Whether image previews appear where expected.

No public links or payment

There is no public download link and no payment link on this website. Release notes, support details, and availability information will be updated before a broader release.
